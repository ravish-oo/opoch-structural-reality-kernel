## TECHNICAL IMPLEMENTATION

### Libraries/Tools:
- **Lucide React** for icons (https://lucide.dev)
- **Framer Motion** for animations (scroll-triggered)
- **Tailwind** for styling + gradients
- **Radix UI** or **shadcn/ui** for card components

### Key CSS/Effects:
```css
/* Timeline line */
.timeline {
  border-left: 2px solid;
  border-image: linear-gradient(to bottom, blue, purple, cosmic) 1;
```
